# movie-recommendation-system

## requirements
- Node.js
- python
- pip

## Frontend
```bash
cd frontend
npm install
```

## Backend

### For Linux
```bash
cd backend
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### For Windows
```bash
cd backend
python -m venv .venv
source .venv\Scripts\activate
pip install -r requirements.txt
```

after installing any dependencies for your own use. Use this command to save the dependencies for others.
### For Windows
```bash
cd backend
source .venv\Scripts\activate
pip freeze > requirements.txt
```
