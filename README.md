# 👥 Team Members & Responsibilities

This project was developed collaboratively by a team of four members, with each member contributing to a specific part of the system.

| Team Member | Role | Responsibilities |
|-------------|------|------------------|
| **MD ASIF** | **AI/ML Team Lead** | Led the AI/ML development, selected the MovieLens 20M dataset, performed data preprocessing, exploratory data analysis (EDA), feature engineering, implemented Content-Based Filtering (TF-IDF + Cosine Similarity), Collaborative Filtering, Hybrid Recommendation System, evaluated model performance, and coordinated AI/ML integration with the application. |
| **Member 2** | **Backend Developer** | Developed backend APIs, integrated the recommendation engine with the application, managed server-side logic, handled API requests/responses, and connected the frontend with the recommendation model. |
| **Member 3** | **Frontend Developer** | Designed and developed the user interface, implemented movie search, recommendation pages, responsive layouts, and improved the overall user experience. |
| **Member 4** | **Database & Deployment Engineer** | Managed the database, handled data storage, deployment, application hosting, testing, version control, and deployment configuration. |
